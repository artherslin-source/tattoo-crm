# 安裝與設定

本目錄提供客戶安裝時使用的工具。

## 替換網址.js

**用途**：將系統內所有開發/示範用網址一鍵替換為客戶自己的後端與前端網址，避免上線後請求誤導向開發站。

**使用方式**（請在專案根目錄執行）：

```bash
node 安裝與設定/替換網址.js
```

依提示輸入「後端 API 網址」與「前端網站網址」後，腳本會更新：

- 前端：`next.config.ts`、`src/lib/api.ts`、預約公開頁、`.env.local.example`
- 後端：`.env.example` 的 CORS 設定

完成後請依「系統安裝上線說明」建立實際的 `.env` / `.env.local` 並重新建置。
