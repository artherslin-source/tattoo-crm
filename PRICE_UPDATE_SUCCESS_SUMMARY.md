## 🔧 技術實施細節

### 使用工具
- **腳本語言：** Node.js
- **腳本檔案：** `update-color-prices.js`
- **API 調用：** RESTful API (PATCH)

### API 端點
```
POST   /auth/login                           登入
GET    /admin/services                       獲取所有服務
GET    /admin/service-variants/service/:id   獲取服務規格
PATCH  /admin/service-variants/:variantId    更新規格價格
```

### 更新欄位
```json
{
  "priceModifier": 價格數值
}
```

### 數據結構
```javascript
const PRICE_DATA = {
  '服務名稱': {
    '割線': 價格,
    '黑白': 價格,
    '半彩': 價格,
    '全彩': 價格
  }
};
```
