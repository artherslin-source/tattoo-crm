## 🔧 技術細節

### 實施方式

**工具：** Node.js 腳本（`update-color-prices.js`）

**API 調用：**
- 登入：`POST /auth/login`
- 獲取服務：`GET /admin/services`
- 獲取規格：`GET /admin/service-variants/service/:serviceId`
- 更新價格：`PATCH /admin/service-variants/:variantId`

**更新欄位：** `priceModifier`

**數據映射：**
```javascript
const PRICE_DATA = {
  '服務名稱': { 
    割線: 價格, 
    黑白: 價格, 
    半彩: 價格, 
    全彩: 價格 
  },
  ...
};
```
